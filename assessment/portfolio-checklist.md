## Format Requirements

Before submitting, verify that your portfolio meets the following formatting standards:

- [ ] **Consistent headers** — Each artifact has a clear title and is easy to locate within the portfolio.
- [ ] **Professional formatting** — Clean layout, readable fonts, appropriate use of tables and lists; no raw markdown or broken formatting in the PDF.
- [ ] **Case context clearly identified** — Your selected case context is stated on the cover page and referenced throughout each artifact where relevant.
- [ ] **AI assistance noted in footer** — If you used AI tools (e.g., ChatGPT, Claude, Copilot) in preparing any artifact, note this in the document footer. State which tools were used and for what purpose. This is a transparency requirement, not a penalty.

## Self-Check Questions

Before submitting each artifact, ask yourself the following questions. If you cannot answer "yes" to all three for every artifact, revisit that section.

**For each of the 17 artifacts:**

1. **Is it complete?**
   Does the artifact address all required elements? Are there any blank sections, placeholder text, or "TODO" notes remaining?

2. **Is it specific to your case context?**
   Would a reader know which case context you are working with based on the artifact alone? Does it reference the specific constraints, team composition, organizational setting, and business challenges of your chosen case?

3. **Would a reader understand it without explanation?**
   If someone outside this course read this artifact, would they understand what it is, what decisions were made, and why? Is it self-contained and clearly written?
